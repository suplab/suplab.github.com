How do you put two divs side by side?
It is really simple and there are different ways to do it, but personally I prefer to use a certain property of CSS called float. The float property has two values, namely left and right.
And here is how it is achieved:

The Code

In your css create 2 divs and call them, say, lefty and righty {% highlight css lineos %} #lefty{ float:left; background:#ff0000; color:#ffffff; width:130px; } #righty{ float:right; background:#0000ff; color:#ffffff; width:130px; } {% endhighlight %} In your HTML file just call the two divs and use them, as you wish.
